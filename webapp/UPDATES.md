## 🎯 Функционал реферальной системы

### Как работает:
1. Пользователь получает уникальную ссылку вида: `https://t.me/StarPayUzBot?start=ref_USER_ID`
2. Делится ссылкой с друзьями
3. Когда друг регистрируется по ссылке, пользователь получает 10% от всех покупок друга
4. Статистика отображается в реальном времени

### API endpoints:
- `GET /api/bot/referrals/:userId` - получить статистику рефералов
- Возвращает:
  - `total_referrals` - количество приглашенных
  - `total_earnings` - общий заработок
  - `referrals[]` - список рефералов с деталями
