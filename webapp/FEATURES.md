# StarPayUz WebApp - Новые функции ✨

## 🌍 Мультиязычность
- **3 языка**: Узбекский, Русский, Английский
- Переключение через страницу настроек
- Автоматическое сохранение выбранного языка
- Все тексты переведены

### Как использовать:
```javascript
// В любом месте приложения
i18n.t('key_name') // Получить перевод
i18n.setLanguage('ru') // Изменить язык
i18n.getCurrentLanguage() // Текущий язык
```

## ⚙️ Страница настроек
**URL:** `/pages/settings.html`

### Функции:
- 🌐 Выбор языка (Узбекский/Русский/Английский)
- 🔔 Включение/выключение уведомлений
- ℹ️ Информация о приложении
- 💬 Связь с поддержкой

### Дизайн:
- Красивые карточки настроек
- Модальное окно выбора языка с флагами
- Toggle switches для переключателей
- Иконки для каждой настройки

## 📊 Страница статистики
**URL:** `/pages/stats.html`

### Функции:
- 📈 Рейтинг покупателей (топ-10)
- 🏆 Медали для топ-3 (золото, серебро, бронза)
- 📅 Фильтры: Сегодня / Эта неделя / Этот месяц
- 💰 Общая сумма продаж
- 👥 Количество пользователей

### Дизайн:
- Красивые градиенты для медалей
- Анимации при наведении
- Адаптивная сетка
- Иконки и эмодзи

### API Endpoint:
```javascript
GET /api/bot/stats?period=today|week|month

Response:

```

## 🔧 Технические детали

### Новые файлы:
```
webapp/
├── js/
│   └── i18n.js              # Система мультиязычности
├── pages/
│   ├── settings.html        # Страница настроек
│   └── stats.html           # Страница статистики
└── FEATURES.md              # Этот файл
```

### Зависимости:
- Font Awesome 6.4.0
- Telegram WebApp SDK
- LocalStorage для сохранения настроек

### LocalStorage ключи:
- `app_language` - Выбранный язык (uz/ru/en)
- `notifications_enabled` - Включены ли уведомления

## 🚀 Деплой
Все изменения автоматически деплоятся при пуше в репозиторий.
