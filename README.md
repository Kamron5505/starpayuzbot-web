# StarPayUz Web App

Веб-приложение для Telegram бота StarPayUz.

## Функции

- 🌍 Мультиязычность (Узбекский, Русский, Английский)
- 💰 Пополнение баланса
- 🏆 Рейтинг покупателей
- ⚙️ Настройки пользователя
- 📱 Адаптивный дизайн

## Развёртывание на Vercel

1. Подключи репозиторий к Vercel
2. Vercel автоматически развернёт приложение
3. Скопируй публичный URL

## Локальный запуск

```bash
npm install
npm start
```

Приложение будет доступно на `http://localhost:3000`
